The 10 fixes

In order from the review.

  1. Em dash removed from copylaw

    The trips section subhead read "Pick one to use as a starting point — fork it into your cart, then make it yours." The em dash is a banned punctuation per shared design laws. Rewritten with a comma. Bonus: the new line is shorter and lands harder ("Fork one as a starting point, then make it yours").

    file apps/web/app/page.tsx

  2. Pure #000/#fff swapped for OKLCH-tinted neutralslaw

    Every neutral now carries a faint cornflower hue (~250°) and reduces chroma at extremes so it never reads as garish. Body, page tokens, surface tokens, and the tier-badge background all updated. Visually identical to a casual eye; passes the law and reads more cinematic side-by-side.

    --fgoklch(0.985 0.006 250)
    --bgoklch(0.12 0.008 250)
    --surface-1oklch(0.18 0.008 250)
    --surface-2oklch(0.22 0.008 250)
    files apps/web/app/globals.css, apps/web/tailwind.config.ts, apps/web/app/page.tsx

  3. Hero h1 bumped 26/34 to 36/52pull

    The landing now anchors on a real headline. Tracking nudged tighter at the larger size; entrance ease shifted from default to a quart curve over 600ms. Subhead lifted to 15/17 with more breathing room before the SearchBar.

    file apps/web/app/page.tsx

  4. Two-hero problem resolvedpull

    Removed the "Fan-voted favorites" eyebrow stack. The section heading is now a single line, "Steal a trip", with a one-sentence support line. The page reads with one clear visual spine: hero, search, grid.

    file apps/web/app/page.tsx

  5. Grid capped at xl:6pull

    Was 2/3/4/5/6/7 columns. The 7-col tier squashed cards into postage stamps where the photo lost its job. New cap is 6 at xl. Image height bumped 32 to 36 to compensate. Loading skeleton shows 12 (was 14).

    file apps/web/app/page.tsx

  6. CTA hierarchy invertedpull

    The pure-white "Get started" pill was visually louder than the actual primary action (the Search button inside the SearchBar). Get started is now an outlined ghost pill (border, transparent fill); Search keeps its accent fill. The user's eye now lands on the right CTA.

    file apps/web/app/page.tsx

  7. Hero gradient made deliberatepolish

    Was a near-flat #141414 to #050505 linear gradient that read as accidental. Now a single off-center cornflower radial spotlight at top-right over a tinted near-black base. Adds depth and signals the accent without committing more than 10% surface to the color.

    file apps/web/app/globals.css

  8. Mobile nav drawer addedpolish

    The navbar previously was hidden md:flex with no fallback. Added a hamburger that toggles a drawer beneath the sticky header. Drawer carries the same NAV_LINKS plus Sign in and Get started. Entrance uses AnimatePresence with a 160ms ease-out slide. Active-route highlighting works in both desktop and drawer.

    file apps/web/app/page.tsx

  9. Skeleton placeholders unified to white/8polish

    Image placeholder bumped from bg-white/5 to bg-white/8 to match the text-row stubs. The skeleton now pulses as a single coherent shape.

    file apps/web/app/page.tsx

  10. Active-state for nav linkspolish

    Extracted a small NavLink component that uses usePathname() to detect the current route. Active route gets a bg-white/10 pill with full-opacity text. Applies to both the desktop nav and the mobile drawer.

    file apps/web/app/page.tsx

Other small wins

Edits that fell out of the same pass.

  1. Tier badge upgraded

    Was bg-white/90 text-accent (a white blob with blue text on the image). Now bg-[oklch(0.12_0.008_250)]/85 with a thin white/10 hairline and text-accent-light label. Matches the brand and reads as part of the dark system rather than a bright sticker.

  2. Trip price color shifted to accent-light

    text-accent on the dark card was readable but heavy. text-accent-light at #7BA3F4 sits more comfortably on the surface and keeps the per-card pop without screaming.

  3. Card border separator uses brand-aware rule

    The within-card divider border-[rgba(91,141,239,0.05)] was almost invisible on the new dark surface. Now border-white/8, consistent with the rest of the system.
